# Análise de Dados Logísticos com Python

Projeto de análise de desempenho logístico utilizando Python e Pandas.

## Objetivo
Analisar dados de entregas para identificar padrões de desempenho e gargalos operacionais.

## Análises realizadas
- Cálculo de tempo médio de entrega
- Cálculo de custo médio de frete
- Taxa de atraso
- Análise por cidade
- Identificação de regiões com problemas logísticos

## Tecnologias utilizadas
- Python
- Pandas
- Matplotlib

## Resultados
Foram identificadas variações relevantes entre cidades, incluindo regiões com maior tempo de entrega e maior custo, indicando possíveis ineficiências operacionais.
